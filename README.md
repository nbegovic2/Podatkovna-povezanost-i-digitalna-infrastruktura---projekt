# Podatkovna-povezanost-i-digitalna-infrastruktura---projekt

## Bucket list

Samostalni web projekt izrađen u svrhu kolegija Podatkovna povezanost i digitalna infrastruktura.
Stranica omogućuje korisniku vođenje osobnog bucket lista putnih destinacija — dodavanje mjesta koja želi posjetiti, filtriranje po kontinentima te označavanje destinacija kao posjećenih.

## Korištene tehnologije

**HTML5:** Struktura stranica  
**CSS3:** Responzivni vizualni dizajn  
**JavaScript:** Interaktivnost i logika aplikacije  
**JSON:** Baza početnih destinacija   
**XML:** Podaci o kontinentima   
**LocalStorage:** Pohrana korisničkih unosa u pregledniku  

## Stranice

**Početna (index.html)** — forma za dodavanje destinacija, statistike i progress bar  
**Destinacije (destinacije.html)** — prikaz svih destinacija s filterima po kontinentu  
**Kontinenti (kontinenti.html)** — informacije o kontinentima učitane iz XML datoteke  
**Kontakt (kontakt.html)** — kontakt informacije  


## Kako pokrenuti projekt

Preuzmi ovaj repozitorij  
Otvori index.html u svom pregledniku
